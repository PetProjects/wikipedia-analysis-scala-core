Wikipedia history dates API

Available REST API history dates queries.

  • /v1/wikiq/tokens/{date-range, e.g. "50bc-50ad"}
    Queries top N (default count is 10) tokens for the specified dates period
  • /v1/wikiq/articles/{date-range, e.g. "50bc-50ad"}
    NOT IMPLEMENTED
  • /v1/wikiq/distribution/{token, e.g. "rome"}
